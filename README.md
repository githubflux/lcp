## Installation

1. Prerequisite

   - python3
   - pip3

2. Clone the repository.

   ```bash
   git clone https://gitlab.com/guard-repositories/lcp.git
   cd lcp
   ```

3. Install the dependencies.

   ```bash
   pip3 install -r requirements.txt
   ```

## Usage

### Display help

```bash
python3 main.py -h
```

### Execute

```bash
python3 main.py
```
